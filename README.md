## 🌟 Project Overview

The **Student Management System** is a modern web-based platform designed to streamline academic and administrative activities within an educational institution.

The system provides dedicated dashboards for:

🎓 Students
👩‍🏫 Faculty
🏢 Head of Department (HOD)
🎓 Principal
📝 Controller of Examination (COE)
👨‍👩‍👧 Parents

This platform enables efficient monitoring, communication, performance analysis, attendance tracking, and academic management through role-based dashboards.
